🎵 Music Library
Full filesystem path to the music directory on the VPS.
On startup, Chilled Koala loads metadata from .library-cache.json — instant, no filesystem scan.
The cache is invalidated automatically when the app version changes (schema version mismatch) or the library path changes.
Run Rescan & Rebuild Cache after adding, removing, or re-tagging FLAC files.
🔴 GO LIVE — AzuraCast DJ Connection
Chilled Koala connects here when you press GO LIVE. This goes to Liquidsoap (not Icecast directly), which is what allows it to override AutoDJ.
AzuraCast → Your Station → Live Streaming → Streamers/DJs → Connection Information
Credentials (username & password) are entered each time you press GO LIVE — they are your AzuraCast DJ account credentials, same as your Chilled Koala login.
📡 Icecast — Listener Output & Metadata
This is where listeners connect and where track metadata is sent. Not used for GO LIVE.
All values are saved to config.ini on the server.
🕐 Clock Synchronisation
The track timer accuracy depends on both VPS and PC being synchronised to NTP. If NTP is not active on the VPS, the timer can drift by several seconds. PC browser time is set by your operating system — ensure it is also NTP-synced.
PC clock: run the following in an Administrator PowerShell to sync to the same NTP source as the VPS:
w32tm /resync /force  —  if that fails:   Restart-Service w32tm ; w32tm /resync /force
🎙 DJ Mic Delay Compensation (Loc Mic 1 & 2)
The DJ hears PGM1 through earphones with a delay of network latency + 700ms jitter buffer. They speak to what they hear, so their voice arrives at the server early relative to the music.
Setting this delay aligns DJ voice with music in the Icecast broadcast output.
Auto-Detect calculates the value from the measured network RTT. Click Apply to save.
0ms
RTT not yet measured — open Settings after login completes.
Start with Auto-Detect, then fine-tune: increase if voice sounds early in the broadcast, decrease if late.
0ms = compensation off. Value is saved to config.ini and restored on restart.
